# Set Leaderstat

## Description

Sets the given value of a players leaderstat.

#### Available to

* [x] Scripts
* [ ] LocalScripts
* [ ] Yields

## Inputs

#### Player (Object)

The player that you want to set the leaderstat.

* [x] Can use Variable

```
game.Players.Roblox
```

#### StatName (String)

The name of the stat that you want to set.

* [x] Can use Variable

```
Coins
```

#### Value (Number)

The number that you want to set to the leaderstat.

```
50
```
